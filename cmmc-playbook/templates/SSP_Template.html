System Security Plan (SSP)

TANO CMMC Playbook – System Security Plan

1. System Overview

Describe the system, users, applications, and security boundary.

2. CUI Handling

Identify where CUI exists and how it flows through the system.

3. Applied Controls

Summaries of applicable CMMC Level 1 and Level 2 practices.

4. Roles & Responsibilities

List key IT, security, and management roles responsible for this system.

5. Network Diagram

Reference or embed the network diagram that shows the system and CUI boundary.

6. Policies and Procedures

Link or reference relevant policies (access control, incident response, media protection, etc.).

7. Risk Assessment Summary

Summarize the top risks identified, their impact, and mitigation plans.
